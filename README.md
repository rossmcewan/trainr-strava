> Strava integration for Trainr virtual coaching


## Install

```sh
$ npm install --save trainr-strava
```


## Usage

```js
var trainrStrava = require('trainr-strava');

trainrStrava('Rainbow');
```

```sh
$ npm install --global trainr-strava
$ trainr-strava --help
```
